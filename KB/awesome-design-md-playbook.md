# awesome-design-md — Website & Design Building System
## Source: https://github.com/smartvaultco/awesome-design-md
## Purpose: Build pixel-perfect websites and landing pages using AI + markdown design specs

---

## What It Is

A collection of 58+ DESIGN.md files extracted from real production websites (Apple, Stripe, Vercel, Linear, Tesla, etc.). You copy a DESIGN.md into your project, tell your AI agent "build me a page that looks like this" and get pixel-perfect UI.

**No Figma. No JSON configs. Just markdown that LLMs read natively.**

---

## How to Use It

1. Pick a design system from the collection
2. Copy the DESIGN.md to your project root
3. Tell Claude Code / Cursor / any AI agent: "Build this page following DESIGN.md"
4. Get production-ready UI

## Each DESIGN.md Includes
1. Visual theme & atmosphere
2. Color palette & roles
3. Typography rules & hierarchy
4. Component styling (buttons, cards, inputs, nav)
5. Layout principles & spacing scales
6. Depth & elevation systems
7. Design do's and don'ts
8. Responsive behavior guidelines
9. Agent prompt guide (instructions for AI)
