## AUTOMATION STRATEGY (Replace People With Workflows)

### WF-01: Lead Capture & Qualify
- **Trigger:** Form submission / chat widget / phone call
- **Actions:** Tag `LEAD-New` → assign to pipeline → AI Chatbot qualifies → books demo call
- **Bot:** Closer Bot monitors, Automation Bot maintains

### WF-02: Self-Selling Demo Delivery
- **Trigger:** Demo requested (form/chat/DM)
- **Actions:** Send branded Smart Website demo link → AI Voice Agent handles live call → log result
- **Bot:** Closer Bot

### WF-03: 14-Day Free Trial Onboard
- **Trigger:** Prospect accepts trial
- **Actions:** Spin up sub-account → inject AI widgets → activate Voice Agent + Chatbot → start 14-day timer
- **Bot:** Automation Bot

### WF-04: Trial Expiry Close
- **Trigger:** Day 12 of trial
- **Actions:** Send "trial ending" SMS/email → Day 14 deactivate AI → Day 15 send pain-point message: "How many leads are you willing to lose?"
- **Bot:** Closer Bot + Email Bot

### WF-05: Client Onboarding (Paid)
- **Trigger:** Invoice paid
- **Actions:** Welcome email → onboarding form (logo, domain, brand colors) → AI chatbot follows up → activate all workflows
- **Bot:** Automation Bot + Email Bot

### WF-06: Database Reactivation
- **Trigger:** Client uploads old contact list (or tag applied)
- **Actions:** Import contacts → 3-message SMS sequence with 50% off offer → route responses to Closer Bot
- **Bot:** Automation Bot + Closer Bot

### WF-07: Reputation Management
- **Trigger:** Opportunity moved to "Won" stage (or Zapier from Square/Stripe)
- **Actions:** Wait 60 min → send review request SMS → 24h follow-up email → log review
- **Bot:** Email Bot + Automation Bot

### WF-08: AEO/SEO Content Deploy
- **Trigger:** New client onboarded (SEO package)
- **Actions:** ChatGPT generates 76 FAQs → AI writes Schema Markup → embed in site header → copy FAQs to Chatbot KB → schedule 7+ GMB posts/month
- **Bot:** Content Builder + Automation Bot

### WF-09: Cold Outreach Engine (New Clients)
- **Trigger:** Weekly (Monday 9AM)
- **Actions:** ChatGPT scrapes Google Maps for 25 leads in target niche → filter → import CSV to GHL → 3-Tier Blitz: automated SMS/email (Tier 1) → cold call responders (Tier 2) → DM non-responders (Tier 3)
- **Bot:** Hustle Bot + Closer Bot

### WF-10: SaaS Rebilling Passive Income
- **Trigger:** Ongoing (automatic)
- **Actions:** GHL $497 plan auto-marks up client SMS, email, AI, domains, WhatsApp usage
- **Bot:** Analytics Bot tracks margin

### WF-11: Weekly Optimization Loop
- **Trigger:** Sunday 10AM
- **Actions:** Analytics Bot pulls dashboards → identifies weakest conversion point → CEO reviews → single fix shipped → documented
- **Bot:** Analytics Bot + CEO
