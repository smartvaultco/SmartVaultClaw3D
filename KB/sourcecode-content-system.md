## System Overview
Automates tech and spiritual content ecosystem for Ra'el Shodomi. Scales output via AI agents + automation for newsletters, podcasts, product funnels, and mastermind community.

## Three-Agent Architecture

### Agent 1: Newsletter Alchemist
- Draft weekly emails for SmartVault + Ra'el audiences
- Steps: Draft via ChatGPT/NotebookLM → Format in MailerLite with correct banner → Embed podcast/IG/CTA links → Schedule + test → Save to Google Drive + track in Notion
- Tools: ChatGPT, Gemini, MailerLite (Free), Canva, NotebookLM

### Agent 2: Podcast Publisher
- Two podcast properties:
  - **"Decode the Future"** — AI and Technology
  - **"The Invisible Architect"** — Spiritual and Esoteric
- Steps: Generate script (ChatGPT/NotebookLM) → Record audio (Google Veo 3/ElevenLabs) → Upload to Buzzsprout/Anchor.fm → Create promo clips (CapCut/Canva) → Share on IG + embed in newsletter → Archive in Google Drive
- Tools: NotebookLM, ChatGPT, Veo 3, ElevenLabs, Canva, CapCut, Anchor.fm, Buzzsprout

### Agent 3: Vault Monetizer
- Products: Crypto eBook (free lead magnet), SmartVault + Ra'el merch (Printful), Affiliate Program, SourceCode Circle Mastermind
- Email sequence: Day 0 (deliver eBook) → Day 1 (promote merch) → Day 3 (invite to mastermind)
- Steps: MailerLite opt-in form → automation sequence → Canva merch design → Printful connection → Affiliate registration + Google Sheet tracking → Telegram/Discord invite
- Tools: MailerLite, Stripe, Gumroad, Canva, Printful, Telegram, Discord, Google Sheets

## Weekly Execution Checklist
1. Newsletter written and formatted
2. Podcast recorded and uploaded
3. Newsletter sent with links embedded
4. Promo clip created and posted
5. eBook funnel checked
6. Affiliate + mastermind invitations sent
7. All files backed up to Google Drive
