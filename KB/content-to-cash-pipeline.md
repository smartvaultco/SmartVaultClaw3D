## QUICK-START: RUN THE PIPELINE FOR A NEW TOPIC

### Day 1: Research (Steps 1-2)
1. Open [creator-brief.md](creator-brief.md) — update if needed for new topic
2. Run 10 ATP searches per [atp-research-guide.md](atp-research-guide.md)
3. Collect data from Reddit, YouTube, Quora, Ad Libraries
4. Select Top 10 questions (ranked by frequency, intent, pain, authority)

### Day 2: Write (Step 3)
5. Create NotebookLM notebook → upload Creator Brief + all research
6. Generate 10 chapter outlines via NotebookLM
7. Write 10 chapters via Claude (use prompt templates from autopilot-workflow.md)
8. Generate 10 course scripts via Claude

### Day 3: Produce (Step 4)
9. Layout ebook in Canva → export PDF
10. Record course videos (screen share + talking head)
11. Create worksheets in Canva → export PDFs
12. QA everything through NotebookLM upload

### Day 4: Deploy (Step 5)
13. Build GHL pages (lead magnet, tripwire, sales page, membership)
14. Upload ebook + course to GHL
15. Write + activate 7-day email nurture via Claude
16. Test full funnel end-to-end (opt-in → delivery → purchase → access)

### Day 5: Launch
17. Post first organic content (LinkedIn + YouTube + Reels)
18. Email any existing contacts
19. Start paid ads after 50+ organic leads prove the message
