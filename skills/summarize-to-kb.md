# Summarize Content to Knowledge Base

Convert any content (YouTube transcript, article, podcast notes, course content) into a structured KB file for Smart Vault Co.

Ask the user:
1. What is the source? (YouTube channel, course name, article URL)
2. What topic/category? (trading / crypto / real estate / business / ai-tools)
3. Who is the source? (e.g., Sean Terry, John Brown, Alex Hormozi)

Then:
1. Process the pasted content
2. Extract: key concepts, actionable steps, scripts, frameworks, rules, and tools mentioned
3. Structure it as a clean markdown KB file with headers
4. Save to the correct KB subfolder: `e:/Smart-Vault-Ai/KB/[category]/[source-name]-[topic].md`
5. Note which bots should reference this file in their MEMORY.md or TOOLS.md

Output: confirm file saved + one-line summary of what was captured.
