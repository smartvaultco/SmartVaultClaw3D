## LANDING PAGE STRUCTURE (From GHL Mastery)

### Funnel vs Website
Funnels are step-by-step paths for ONE action (buy/book). NOT exploratory websites.

### Optimal Lead Magnet Funnel (4 Steps):
1. **Opt-in Page** — High-powered lead magnet to capture email
2. **Sales Page** — Educates and sells core offer
3. **Checkout Page** — Value stack, scarcity elements, price anchor
4. **Order Confirmation** — Thank you + next steps

### Key Page Elements:
- Simple, clear headline at top
- Short video explaining how you solve their problem
- Calendar element for booking (if service-based)
- Case studies/testimonials below for trust
- Bonuses listed clearly
- Strong, clear CTA button

### Builder Tips:
- Use **Prebuilt Sections** ("+" icon → Prebuilt Sections) for CTA, FAQ, Footer blocks
- **Mobile Mode** (phone icon at top) — most traffic is mobile
- Center elements, booking button front-and-center
- No excessive copy or animations on mobile

## GHL AI & CONVERSION FEATURES

### Split Testing
- Clone a funnel step to run A/B split tests
- Track stats to see which variation converts better

### AI Chatbot (Conversational AI)
- Sites > Chat Widget → customize
- Funnel Settings → toggle Chat Widget on
- AI Agents > Conversational AI → build bot for 24/7 booking
- Enable **Auto follow-up** in bot settings (re-messages leads who stop responding)
- **Trigger a workflow** from bot — fires SMS on info capture

### AEO (Answer Engine Optimization)
- Generate FAQ Schema Markup via ChatGPT
- Paste into Funnel Settings > Tracking Code (Header section)
- Helps rank on Google SGE, ChatGPT, Claude AI overviews

## AUTOMATION WORKFLOWS

### Speed to Lead — "Fast Five" Blitz
- Email + SMS + Call within first 5 minutes of opt-in
- Use GHL's pre-built "Fast Five" recipe

### Pipeline Automation
- Trigger: Form Submitted / Order Form Submission
- Action: Create/Update Opportunity → Pipeline + Stage
- Auto-update stages as leads open emails / reply

### Sequences
- Wait blocks (1 day, 3 days, etc.)
- If/Else conditional logic for segmentation
- Reply "no" → different branch
- No reply → follow-up sequence

### Abandoned Cart Recovery
- Trigger: left checkout without buying
- Wait → Email/SMS sequence to pull back

### No-Show Nurture
- Trigger: Appointment Status = "no-show"
- Fire email + SMS to rebook

### Trigger Links
- Marketing > Trigger Links → custom URLs
- Put in SMS/email campaigns
- Trigger Link Clicked → fires new automation
