### Setup Steps in GHL:

1. **Go to:** Sites → Membership → Create New

3. **Access:** Paid (connected to Stripe products below)

### Course Structure in GHL:

**Category 1: Welcome**
| Lesson | Content | Attachments |
|--------|---------|-------------|
| Welcome Video | 5 min intro: what to expect, how to use the course | Course roadmap PDF |
| BONUS: Smart Vault Setup | NotebookLM walkthrough (from Module 10 content) | Setup checklist PDF |

**Category 2-11: Modules 1-10**

For EACH module (repeat this structure 10 times):

| Lesson | Content | Attachments |
|--------|---------|-------------|
| Module [#] Video Lesson | Upload recorded video (15-20 min, use scripts from COURSE-FULL.md) | None |
| Module [#] Worksheet | Text lesson with worksheet content embedded | Worksheet PDF download |
| Module [#] Action Checklist | Text lesson with checklist | Checklist PDF download |
| Module [#] Quiz | Use GHL's built-in quiz/survey feature or embed quiz questions | None |

  - + 1:1 Audit ($497): All modules + community + calendar booking link for audit
- **Completion Tracking:** Enable progress tracking so students see % complete
- **Offer Wall:** After module 10, show upsell to Audit Call or Retainer
