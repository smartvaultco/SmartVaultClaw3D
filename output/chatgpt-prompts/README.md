# ChatGPT Deep Research Prompt Bundles
## How to Use

Each file below contains 1 mega-prompt per KB category. For each one:

1. Open ChatGPT (GPT-4o or GPT-4o with Deep Research)
2. Paste the ENTIRE prompt into one message
3. Let it run — each prompt asks ChatGPT to produce a comprehensive research document
4. Copy the output and save it to `e:/Smart-Vault-Ai/output/deep-research/chatgpt-[category].md`
5. Upload to the matching NotebookLM notebook as an additional source

## Why 3 LLMs?

Each AI has different training data, reasoning patterns, and blind spots:
- **Claude** (already done) — strongest on strategy synthesis, frameworks, structured analysis
- **Gemini** (NotebookLM) — strongest on cross-referencing sources, finding contradictions, grounding in citations
- **ChatGPT** (these prompts) — strongest on web-connected research, current data, creative angles, gap-filling

The "Multitude of Counselors" approach: compare all 3 outputs, keep what aligns, investigate what conflicts, and fill gaps no single AI caught.
